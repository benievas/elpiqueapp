# ElPiqueApp · Player UI Kit

Mobile-first PWA for players who want to book a court. Recreates the Home hero slider, Explorar, Complejo detail with booking widget, and the bottom nav — all using liquid glass + lime neon aesthetic.

## Components
- `PhoneFrame.jsx` — iPhone-style frame
- `BottomNav.jsx` — fixed glass nav with 6 items
- `HomeScreen.jsx` — hero slider, stats, featured complejos
- `ExplorarScreen.jsx` — search + filter chips + grid of cards
- `ComplejoScreen.jsx` — gallery, info, court list, booking CTA
- `BookingModal.jsx` — date/hour/duration booking flow with WhatsApp pre-fill

## Source
Recreated from `MatchPro/src/app/page.tsx`, `MatchPro/src/app/explorar/page.tsx`, `MatchPro/src/app/complejo/[slug]/page.tsx`, and `MatchPro/src/components/BottomNav.tsx`.
